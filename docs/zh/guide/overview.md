# 项目概述

## 这是什么？

**微信每日天气推送** 是一个自动化系统，每天定时向微信推送包含 **天气信息、AI 点评、英语情话** 的精美消息。

## 功能特色

| 特性 | 说明 |
|---|---|
| 🌤️ 多源天气 | 和风天气 API（主），中国天气网（备用自动切换） |
| 🤖 AI 生成 | Gemini 根据实时天气生成中文点评 + 英语情话 |
| 📱 微信推送 | PushPlus 一对多推送，支持群组 |
| 🏙️ 多城市 | 可配置任意数量城市，每个城市对应一个人 |
| ⏰ 早晚双推 | 早安 07:30 + 晚安 22:00，自动区分模式 |
| ☁️ 免服务器 | GitHub Actions 定时运行，完全免费 |

## 数据流

```
和风天气 API ──→ weather.py ──→┐
                               ├──→ main.py ──→ push.py ──→ PushPlus ──→ 📱 微信
Gemini API ───→ love_message ──┘
                  .py
```

## 推送卡片样式

推送的消息是一张精美的 HTML 卡片，包含：

1. **标题** — "今天也是爱你的一天" + 日期
2. **模式标识** — 🌞 早安推送 / 🌙 晚安推送
3. **各城市天气** — 温度、体感、湿度、风力、日出日落、紫外线
4. **Gemini 点评** — AI 根据天气生成的中文关怀
5. **英语情话** — AI 生成的个性化情话
6. **随机问候语** — 早安/晚安问候
